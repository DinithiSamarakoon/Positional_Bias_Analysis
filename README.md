NLP Project: Yelp Positional Bias Pipeline

This project explores positional bias in LLM summarization using Yelp reviews. It builds a processed dataset, generates positional variations (top/middle/end), and evaluates multiple LLMs across different providers.

🚀 Features
Yelp dataset filtering & preprocessing
Creation of positional review concatenations (Top / Middle / End)
LLM-based summarization experiments
Multi-provider support:
Groq
OpenAI
Together AI
Hugging Face
Mock model (baseline)
Evaluation of positional bias impact
Visualization of results (recall, latency, output differences)
